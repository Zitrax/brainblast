To run the Kyra demo:

Windows
	From the command line, in the KyraBinWin directory, type "krdemos".
	Or browse directory in windows explorer and double click on "krdemos".

Linux
	From the command line, in the KyraBinLinux directory, type "./krdemo".

The demo screens will automatically switch, but you can press spacebar to advance to the next test.

Note that the -f option will run the demo full screen.

See www.grinninglizard.com/kyra for *much* more information.
